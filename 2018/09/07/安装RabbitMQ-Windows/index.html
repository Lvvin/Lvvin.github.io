一 下载安装文件

Rabbit MQ 是建立在强大的Erlang OTP平台上，因此安装Rabbit MQ的前提是安装Erlang。通过下面两个连接可以下载安装最新的版本：

  1. 下载并安装 Eralng OTP For Windows Erlang

  2. 下载并安装Rabbit MQ Server Windows RabbitMQ

百度知道-具体的安装步骤可以参考：

二 配置

  1. 安装后查看环境变量
  1. 启动RabbitMQ

  1. 访问Web管理界面

Rabbitmq管理界面开启

  1. 创建管理员用户

    默认登录时guest 我们需要自己创建自己的

  2. 查看所有用户 >rabbitmqctl.bat list_users

  3. 添加用户 >rabbitmqctl.bat add_user lin Geeklin747496
  4. 设置用户标签 >rabbitmqctl.bat set_user_tags lin adminstrator
  5. 设置用户admin权限 >rabbitmqctl.bat set_permissions -p / lin “.“ “.“ “.*”
  6. 检查 >rabbitmqctl.bat list_users
